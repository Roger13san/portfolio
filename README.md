# Portfólio — Roger Santana

Portfólio pessoal desenvolvido com React + Vite. Disponível em [rogersantana.me](https://rogersantana.me).

## Stack

- React 19
- Vite
- CSS Modules
- Lucide React

## Seções

- **Hero** — apresentação e links principais
- **Sobre mim** — trajetória e perfil
- **Tecnologias** — scroll infinito com stack
- **Projetos** — projetos em destaque com modal de detalhes
- **Trajetória** — timeline de experiência e educação
- **Contato** — canais de contato

## Rodando localmente

```bash
cd Portfolio_project
npm install
npm run dev
```

## Build

```bash
npm run build
```
